# 1. 建立遊戲視窗
```python
import pygame # 導入函式庫

pygame.init() # 初始化pygame

window_x = 800 # 設定遊戲視窗大小
window_y = 600
window = pygame.display.set_mode((window_x, window_y))

pygame.display.set_caption("Pygame貪食蛇") # 定義視窗名稱
```
